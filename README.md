# AI Task Planner

AI Task Planner — хакатонный бот-планировщик задач для студентов. Он принимает
задачи обычным текстом, уточняет недостающие данные, валидирует ввод, сохраняет
задачи в JSON и показывает план по датам и приоритетам.

## Тема и целевая аудитория

Тема проекта: умный AI-помощник для учебного планирования.

Целевая аудитория:

- студенты, которым нужно быстро записывать учебные и бытовые задачи;
- участники командных проектов и хакатонов;
- пользователи, которым удобнее писать задачу естественным языком, а не
  заполнять форму вручную.

Бот помогает превратить фразу вроде `завтра сделать лабу по вебу на 2 часа,
срочно` в структурированную задачу с названием, дедлайном, длительностью,
категорией и приоритетом.

## Возможности

- пошаговый диалог создания задачи;
- извлечение полей задачи через LLM и резервный парсер;
- кнопки `Мой план`, `Подсказка`, `Действия`, `Новая задача`, `Архив`;
- план по датам с сортировкой задач по приоритету;
- выполнение, перенос, удаление и восстановление задач;
- предупреждение, если задача часто откладывалась;
- JSON-хранение задач в `data/tasks.json`.

## Установка и запуск через Poetry

Требования:

- Python 3.11+
- Poetry
- OpenAI API key

Команды для Windows PowerShell:

```powershell
copy .env.example .env
poetry install
```

В файле `.env` укажите ключ:

```env
OPENAI_API_KEY=your_key_here
LLM_PROVIDER=openai
OPENAI_MODEL=gpt-4o-mini
```

Запуск Chainlit:

```powershell
poetry run chainlit run app.py
```

Запуск тестов:

```powershell
poetry run pytest
```

Альтернативный запуск без Poetry:

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
chainlit run app.py
```

После запуска откройте локальный URL, который покажет Chainlit в терминале.

## Пример диалога

```text
Пользователь: Здравствуй
Бот: Добрый день! Я AI-планировщик задач. Хотите добавить задачу?

Пользователь нажимает: Новая задача
Бот: Опишите новую задачу обычным текстом.

Пользователь: Решение задач по математике
Бот: Сколько времени она займёт? Например: 2 часа или 30 минут.

Пользователь нажимает: 31 минута
Бот: Когда дедлайн? Можно написать: сегодня, завтра, 25.04 или 2026-04-25.

Пользователь нажимает: Завтра
Бот: К какой категории отнести задачу?

Пользователь нажимает: Учёба
Бот: Задача «Решение задач по математике» сохранена!

Пользователь нажимает: Мой план
Бот:
Твой план по дням:

26.04 (вс)
----------------------------------------
- Решение задач по математике
  normal · учёба · 31 мин · дедлайн: 26.04 (вс)

Пользователь нажимает: Действия
Бот: Что сделать с задачами?

Пользователь нажимает: Удалить задачи
Бот: Выберите задачу для удаления

Пользователь нажимает: Удалить: Решение задач по математике
Бот: Задача удалена.
```

## Структура проекта

- `app.py` — входная точка Chainlit.
- `config.py` — настройки окружения и LLM.
- `graph/` — LangGraph-сценарий диалога.
- `llm/` — промпт и парсинг пользовательского текста.
- `ui/` — обработчики Chainlit, кнопки и форматирование сообщений.
- `core/` — сортировка и планирование задач.
- `utils/` — валидация и JSON-хранилище.
- `tests/` — тесты для ключевой логики.
