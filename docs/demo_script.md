# Demo Script

## 00:00-00:30 — старт

Открыть Chainlit и показать приветствие:

`Здравствуй`

Бот приветствует пользователя и предлагает добавить задачу обычным текстом.

## 00:30-01:40 — пошаговое добавление

Пользователь:

`Лаба по вебу`

Бот спрашивает длительность.

Пользователь:

`2 часа`

Бот спрашивает дедлайн.

Пользователь:

`завтра`

Бот спрашивает важность.

Пользователь:

`срочно`

Бот спрашивает категорию, если она ещё не определена.

Пользователь:

`учёба`

Бот сохраняет задачу.

## 01:40-02:30 — быстрый ввод

Пользователь:

`завтра сделать лабу по вебу на 2 часа, срочно`

Бот извлекает поля автоматически и сохраняет задачу или уточняет недостающее.

## 02:30-03:00 — план

Пользователь:

`покажи план`

Бот выводит активные задачи, отсортированные по приоритету и дедлайну.

## 03:00-03:30 — действия

Нажать `Выполнено` для первой активной задачи. Затем показать, что задача исчезла
из активного плана.

Нажать `Перенести` для активной задачи. Бот меняет статус и увеличивает
`postponed_count`.
